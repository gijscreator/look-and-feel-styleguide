# Styleguide Snappthis

Ontwerp en maak met een team een styleguide om te gebruiken voor een opdracht van een opdrachtgever.
Gebruik deze link 
``<link rel="stylesheet" href="https://gijscreator.github.io/look-and-feel-styleguide/styleguide.css">`` om de stylesheet te gebruiken in je project.
